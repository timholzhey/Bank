WeLOAD Version 0.92




Web based Learning Object Authoring and Delivery





WeLOAD based on the dem "Portable Web Server" and RELOAD. It is developed under a Creative Commons-License (have a look at the end of this text).

The Portable Web Server is a webserver with the compenents:

Apache 1.3 (Open Source)

PHP 5.04 (Open Source) und

MySQL 3.23.47 (Open Source)

WeLOAD is a WAMP (Windows-Apache-MySQL-PHP). WAMPs are running in millions of webservers in the internet, and these environments are normally very stable.

Phillip Perkmann from the "Tiroler Bildungsservice" realised this WAMP. It is very compact and can be installed on a single USB memory stick. WeLOAD 0.9 is decompressed 17 MB large. In cooperation with "Tiroler Bildungsservice" the IQSH decided to use this WAMP for development proposes.

The second component used in WeLOAD is the Open Source Content Package Editor RELOAD, developed by Bolton University, England. RELOAD stands for "Reusable ELearning Object Authoring And Dilivery". RELOAD uses international standards for generating Content Packages like SCORM or IMS. The programmers of RELOAD found a very elegant way of creating web based HTML output for Content Packages and allowed us to use parts of their source code for WeLOAD.

WeLOAD is a web based Editor, which is able to generate Content in international standards and to deliver the Content Packages on every computer in a school network via browser.

The teacher can collect digital media and materials (text, pictures, sounds, videos, links and other files) in WeLOAD and arange them into Content Packages, which can be delivered to all computers in the local area network without changing the network configuration.

RELOAD is a fantastic tool with many options. WeLOAD is made for school intranets and has the following features :

  • Capturing content from the students and make this content also available as web based Content Packages in the intranet
  • A password protected interface for the teacher
  • Uploading Content Packages without changes to a webspace and make them available via internet
