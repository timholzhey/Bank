Portable Web Server Utility

for Microsoft Windows

"Portable Web Server" is a utility which allows you to manage a complete Apache/MySQL/PHP environment on harddisk or USB memory stick. The tool rewrites all relevant configuration files automatically. The environment runs on any harddisk, and directory. All processes will be started and ended automatically.

Start: weload-start.exe

Configuration of Apache, MySQL, PHP:

When changing configuration files, please use ALWAYS the template files and not the configuration files themselves:

apache/conf/httpd.conf.template and/or
apache/php/php.ini.template
