Portable Web Server Utility

für Microsoft Windows

"Portable Web Server" ist ein Utility, das es Ihnen erleichtert, eine komplette Apache/MySQL/PHP Umgebung auf Festplatte bzw. USB-Stick zu betreiben. Das Utility schreibt alle relevanten Config-Files automatisch um, sodass die Umgebung unabhängig vom vergebenen Laufwerksbuchstaben des Wechseldatenträgers funktioniert. Außerdem werden alle Prozesse automatisch gestartet und beendet.

Start: weload-start.exe

Konfiguration von Apache, MySQL, PHP:

Falls Sie Config-Files ändern möchten, ändern Sie bitte IMMER die entsprechenden Templates und nicht die Config-Files selbst:

apache/conf/httpd.conf.template bzw.
apache/php/php.ini.template
